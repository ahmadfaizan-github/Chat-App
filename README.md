# Chat-App
How to run the project?
Step 1: Download the source code and paste it inside htdocs folder of your XAMPP.
Step 2: Start your Apache and MySQL server from the XAMPP control panel.
Step 3: After completing these steps, go to your browser and open this URL localhost/phpmyadmin, create a new database name with a chatapp.
Step 4: After creating the database, there you can see an import option, just click on that and import the SQL file which is in the ChatApp folder.
Step 5: Everything is done now, just open this URL localhost/chatapp thatâ€™s it. Your chat application is ready to chat.
